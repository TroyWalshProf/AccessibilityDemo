1. Install Chrome

2. Install npm

3. Open a command line and navigate to this folder

4. On the command line run:
npm install testcafe axe-core axe-testcafe

5. On the command line run:
testcafe chrome AxeTest.js

6. Note that accessibility errors are reported

For any issue please refer to:
https://devexpress.github.io/testcafe/documentation/getting-started/
